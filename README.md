# go-springdata-h2-jpa

## h2 
[console-url](http://localhost:8080/h2)
* NB update JDBC URL in order to connect to db, see below
  `jdbc:h2:file:./database/Products`

## /api/products
* get: get product by id
* post: create product
* put: update product
* delete: delete product
